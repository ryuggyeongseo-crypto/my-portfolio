제작 과정

기획 → 레퍼런스 수집 → 무드보드 → 스토리보드 → 스타일 프레임 → 모션 작업 → 편집 작업 → 사운드 작업

색상 디자인
이전 작품
SWIPER 스크롤 자동으로 넘어감(메인)
다음 작품

단순한 시각적 요소가 아닌 감정과 분위기를 전달하는 언어
콘텐츠의 목적과 메시지에 맞는 색채를 선택하고 시청자가 자연스럽게 감정을 받아들일 수 있도록 디자인합니다.

사운드 구성
이전 작품
SWIPER 스크롤 자동으로 넘어감(메인)
다음 작품

영상의 완성도를 결정하는 중요한 요소
BGM, Dialogue, SFX, Ambience의 균형을 통해 장면의 감정선을 강화하고
컷의 전환과 움직임에 맞춘 사운드 연출로 시청자의 몰입도를 높이는 것을 중요하게 생각합니다.

기존에 있던 숏폼 콘텐츠에 적용해보며
편집 역량을 발전 시키고 있습니다.

상세페이지 디자인

실무 역량을 확장하고 있습니다.
